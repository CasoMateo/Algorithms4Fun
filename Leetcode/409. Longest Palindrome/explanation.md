**Algorithm Idea**

Find the frequency of each item in the list. 
Account for even number of occurrences. 
Set the current frequency to either 0 or 1.
Afterwards, create a "center" using only 
one of the remaining "1s" in the list.

**Complexities**

Time: O(N). 
Space: O(N).
