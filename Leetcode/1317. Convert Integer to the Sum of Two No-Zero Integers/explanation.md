**Ailgorithm Idea**

Use a temporary variable to find a number 
that does not contain the digit zero. Furthermore, 
check if the difference between the original number 
and the updated one doesn't contain the digit zero either. 

**Complexities**

Time: O(N).
Space: O(1).
