**Algorithm Idea**

Replace each rectangle with its respective ratio. 
Pay attention to ratios that exceed the frequency of 
two and add all possible combinations of such elements 
to a count variable. Return count // 2. 

**Complexities**

Time: O(N).
Space: O(N).
